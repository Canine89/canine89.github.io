    2. python 3.10

01단계 다음 명령어를 실행하여 3.10 버전의 파이썬 가상 환경을 생성할 수 있음. -3.10이 ‘3.10’ 버전으로 가상 환경을 생성하라는 뜻임. 마지막 venv310은 폴더 이름.
