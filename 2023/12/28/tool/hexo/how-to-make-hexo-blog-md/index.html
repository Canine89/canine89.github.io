hexo로 깃 블로그 만드는 방법

12월 28 2023

걍 따라하셈

내용 없음

01단계 https://hexo.io/ko/index.html 헥소 사이트 접속해서 설치하고 서버 실행하삼. 그럼 스타터가 기초 폴더 구조 다 잡아주고 hexo server를 실행하면 이런 결과가 나올 것임.

02단계 블로그를 다 만들었으니 이제 깃허브에 배포하면 됨. 깃허브 레포 이름을 다음과 같이 <깃허브_아이디>.github.io라고 만드셈.

03단계 이제 이 레포를 만든 헥소 블로그 프로젝트 설정값에 입력해주면 됨. 모든 헥소 프로젝트 폴더 루트에는 _config.yml 파일이 있음. 여기서 설정값들을 만지는 것임. deploy 항목에 type이 git인지 확인하고 repo에 배포할 위치의 레포 주소를 입력하면 되는데, 조심해야 할 점은 Personal access tokens (classic)를 깃허브 계정에서 생성해서 입력해야 한다는 것임.

04단계 깃허브에 로그인한 상태에서 오른쪽 위에 있는 자신의 아이콘을 누르면 Setting를 선택할 수 있고, 왼쪽 메뉴에서 가장 마지막에 있는 developer settings 메뉴에서 이 값을 만들 수 있음.

05단계 이미지 따라서 4개 체크하고 만들기. 만든 토큰값은 만든 직후밖에 볼 수 없으니 반드시반드시반드시 복사한다.

06단계 이제 다시 03단계의 repo를 잘 입력하고, 배포를 진행한다. 다음 명령어를 입력하면 배포가 됨.
